Q.2 Write down difference between Declaration and Initialization a variable.

Ans:In variable Declaration assign the name of variable, like var x;
In variable Initialization assign the value of variable which are declared, like var x=3;

Q.3 In how many ways we can define a variable .

Ans: There are ways to define variable.

  1. var
  2. let
  3. const

Q.4 Is variable is case sensitive ?

Ans:Yes, variables are case sensitive.

Q.5 When we declare a variable can we start it with any number?

Ans: When we want to store a value then we have need to declare the variable. A variable is container. Numbers are not allowed as the first character in names of variables.
